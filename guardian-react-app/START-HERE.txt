STEP 1: Open Command Prompt
───────────────────────────

1. Press Windows Key + R
2. Type: cmd
3. Press Enter


STEP 2: Navigate to Project Folder
───────────────────────────────────

Copy and paste this command:

cd "d:\xampp\htdocs\AdvPHP\Guardian Reinsurance\guardian-react-app"


STEP 3: Install Dependencies (First Time Only)
───────────────────────────────────────────────

Run this command:

npm install

⏱️  This will take 2-3 minutes. Wait for it to complete.


STEP 4: Start the Development Server
─────────────────────────────────────

Run this command:

npm start

🚀 Your browser will automatically open to http://localhost:3000

PROBLEM: "npm is not recognized"
SOLUTION: Install Node.js from https://nodejs.org/
          Choose the LTS (Long Term Support) version

PROBLEM: "Port 3000 is already in use"
SOLUTION: React will automatically use port 3001
          Or stop the other process using port 3000

PROBLEM: Images not showing
SOLUTION: Images are already copied to public/images/
          If missing, run: setup-images.bat

PROBLEM: White screen appears
SOLUTION: Check browser console (F12) for errors
          Make sure npm install completed successfully
